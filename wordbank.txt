# Imposter word bank
# ------------------
# One entry per line. Format:
#   Word | Tag1, Tag2 | Alias1, Alias2
#
# - Word: what the non-imposters actually see for the round.
# - Tags: one or more categories (comma-separated). A word shows up for a
#   round whenever the host's chosen category matches any of its tags —
#   e.g. "Turkey | Animals, Food and Drinks" can appear under either
#   category. Add a brand-new tag and it automatically becomes a
#   selectable category — no other code changes needed.
# - Aliases (optional, third column): alternate spellings/names that also
#   count as a correct guess for that word. Leave the column out entirely
#   if there are no aliases.
#
# Lines starting with # are comments (handy as section headers) and are
# ignored, as are blank lines.
